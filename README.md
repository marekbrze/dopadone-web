# Dopadone

Prosta aplikacja do zarządzania zadaniami i projektami.

## Uruchomienie

**[Otwórz Dopadone →](https://marekbrze.github.io/dopadone-web)**

## Szybki start

1. Otwórz aplikację — kreator poprowadzi Cię przez konfigurację
2. Wybierz obszary życia (np. Praca, Dom, Zdrowie) i dostosuj konteksty
3. Opcjonalnie dodaj pierwszy projekt
4. Gotowe — Twoja struktura jest gotowa do pracy

## Jak to działa

Organizujesz pracę w hierarchii czterech poziomów:

**Obszar** — główna sfera życia
Przykłady: Praca, Dom, Zdrowie, Finanse

&nbsp;&nbsp;**Podobszar** (opcjonalny) — podkategoria obszaru
&nbsp;&nbsp;Przykłady: Samochód, Ogród, Inwestycje

&nbsp;&nbsp;&nbsp;&nbsp;**Projekt** — konkretne przedsięwzięcie z planowanym końcem
&nbsp;&nbsp;&nbsp;&nbsp;Przykłady: Remont łazienki, Prezentacja Q3, Dieta 30 dni

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Zadanie** — jedna czynność do wykonania
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Przykłady: Kup płytki, Wyślij ofertę, Zaplanuj jadłospis

## Widoki

- **Inbox** — zadania bez przypisanego projektu, punkt startowy dnia
- **Dziś** — kalendarz dzienny z blokami czasowymi i wydarzeniami
- **Agenda** — wielotygodniowy przegląd wydarzeń i bloków
- **Planowanie** — organizacja obszarów, projektów i zadań
- **Robienie** — tryb skupienia na aktywnych zadaniach

## Funkcje

- Priorytety zadań (niski, średni, wysoki)
- Nakład pracy (XS–XL)
- Konteksty zadań (np. Telefon, Komputer, Na mieście)
- Notatki przy zadaniach i projektach
- Automatyczne kopie zapasowe co 5 minut
- Eksport i import danych (z opcjonalnym szyfrowaniem)
- Responsywny design (mobile + desktop)

## Dane i synchronizacja

Domyślnie wszystkie dane przechowywane lokalnie w przeglądarce (IndexedDB). Brak konta, brak chmury — Twoje dane są tylko u Ciebie.

Opcjonalnie możesz skonfigurować synchronizację przez Dexie Cloud w Ustawieniach → Synchronizacja.
