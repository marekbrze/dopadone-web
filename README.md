# Dopadone

Prosta aplikacja do zarządzania zadaniami i projektami.

## Uruchomienie

**[Otwórz Dopadone →](https://marekbrze.github.io/dopadone-web)**

## Jak to działa

Organizujesz pracę w strukturze:

- **Obszary** — główne sfery życia (np. Praca, Dom, Hobbys)
- **Podobszary** — szczegółowe kategorie (np. Samochód, Finanse)
- **Projekty** — konkretne przedsięwzięcia
- **Zadania** — pojedyncze czynności do wykonania

## Funkcje

- Priorytety zadań (niski, średni, wysoki)
- Nakład pracy (XS–XL)
- Konteksty (@dom, @praca, itp.)
- Notatki przy zadaniach
- Automatyczne kopie zapasowe
- Responsywny design (mobile + desktop)

## Dane

Wszystkie dane przechowywane lokalnie w przeglądarce. Brak konta, brak chmury — Twoje dane są tylko u Ciebie.
